# Design System — Shree Saraswati Secondary School

## Colors (OKLCH)
- **Primary (navy)**: oklch(0.29 0.045 260) — deep, warm-leaning navy for headers, nav, footer
- **Primary light**: oklch(0.38 0.05 255) — hover / section backgrounds
- **Accent (green-teal)**: oklch(0.55 0.12 175) — tropical teal from the hills, used for CTAs, highlights, active states
- **Accent dark**: oklch(0.42 0.10 170) — deeper version for hover states
- **Gold (warm accent)**: oklch(0.72 0.13 85) — sparingly for emphasis (badges, stars)
- **Background**: oklch(0.97 0.008 80) — warm off-white, tinted toward amber
- **Background alt**: oklch(0.93 0.012 75) — slightly deeper warm tone for card backgrounds
- **Surface (white)**: oklch(0.99 0.003 80) — near-white with a whisper of warmth
- **Text**: oklch(0.25 0.02 260) — near-black with navy tint
- **Text muted**: oklch(0.45 0.02 255)
- **Border**: oklch(0.85 0.01 75)

## Typography
- **Headings**: Playfair Display (700 for major, 600 for minor)
- **Body**: Noto Serif (400, 700 for bold), excellent Devanagari support
- **Scale**: 0.875 / 1 / 1.25 / 1.563 / 1.953 / 2.441 (major third)
- **Body line length**: 65-70ch max
- **Hierarchy**: Weight + size contrast. No all-caps body text.

## Spacing
- **Base unit**: 8px
- **Section padding**: 80px vertical (64px on mobile)
- **Card padding**: 24-32px
- **Grid gaps**: 24-32px

## Elevation
- **Shadow**: 0 2px 12px oklch(0 0 0 / 0.06)
- **Shadow lg**: 0 8px 32px oklch(0 0 0 / 0.10)
- **Shadow accent**: 0 8px 24px oklch(0.55 0.12 175 / 0.25)

## Border radius
- **Default**: 6px
- **Large**: 12px
- **Pill**: 999px

## Motion
- Transition: 0.35s cubic-bezier(0.22, 1, 0.36, 1)
- Hover lift: translateY(-4px)
- No bounce, no elastic
